# 25-26 I1 EISI - Sujet MSPR TPRE813 --- Résumé complet

## Certification

Certification professionnelle : Expert en Informatique et Système
d'Information (RNCP35584)\
Bloc 3 -- Piloter l'informatique décisionnel d'un SI (Big Data &
Business Intelligence)

Objectif : mettre en place une stratégie Data / BI / Big Data permettant
d'exploiter les données pour la décision stratégique.

------------------------------------------------------------------------

# 1. Compétences évaluées

-   Collecter les besoins en données des directions métiers
-   Structurer les données du SI
-   Définir une stratégie data globale
-   Définir une architecture Business Intelligence
-   Mettre à disposition des données structurées aux métiers
-   Définir une stratégie Big Data (collecte, traitement, exploitation)
-   Proposer des modèles statistiques et de machine learning
-   Produire des visualisations exploitables
-   Définir un référentiel de données
-   Créer un entrepôt de données
-   Garantir la qualité des données
-   Assurer la sécurité et la conformité RGPD

------------------------------------------------------------------------

# 2. Organisation de la MSPR

Durée de préparation : **25 heures**

Travail : - équipe de **4 étudiants** - **5 maximum si groupe impair**

### Soutenance

Durée totale : **50 minutes**

-   20 min présentation
-   30 min questions jury

Jury : - 2 évaluateurs externes

Objectif : démontrer les compétences du bloc.

------------------------------------------------------------------------

# 3. Présentation de l'entreprise

**Electio‑Analytics**

Startup fictive créée en **2017**.

Activité : conseil stratégique pour campagnes électorales.

Clients : - candidats - partis politiques - cabinets de conseil

Mission : produire des analyses prospectives basées sur l'exploitation
de données publiques et privées.

------------------------------------------------------------------------

# 4. Besoin stratégique

Créer un système de **prévision des tendances électorales**.

Horizon : **1 à 3 ans**

Indicateurs possibles :

-   sécurité
-   emploi
-   vie associative
-   population
-   activité économique
-   nombre d'entreprises
-   pauvreté

Objectif : avantage concurrentiel.
