## ⚠️ Notes importantes

1. **INSEE** : Les fichiers Excel ont des métadonnées en en-tête → utiliser `skiprows`
2. **Comptes communes** : Très gros fichiers → télécharger par tranches d'années
3. **Élections** : Déjà téléchargé (2.2 GB dans Bronze)
4. **Onglets Excel INSEE** : Données communales dans onglets `COM_XXXX`
